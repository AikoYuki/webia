SISTEMA DE RESERVAS - INSTRUCCIONES

DESCRIPCIÓN:
Sistema simple de reservas para curso educativo.

ARCHIVOS INCLUIDOS:
- index.html          (Formulario principal de reservas)
- reservar.php        (Procesa las reservas)
- ver_reservas.php    (Muestra reservas existentes)  
- config.php          (Configuración de base de datos)
- test_conexion.php   (Para probar conexión a BD)

INSTALACIÓN:
1. Crear base de datos MySQL con la tabla 'reservas'
2. Configurar credenciales en config.php
3. Subir archivos al servidor web
4. Acceder a index.html

FUNCIONALIDAD:
- Formulario con validación básica
- Control de fechas duplicadas
- Listado de reservas existentes
- Navegación entre formulario y listado

NOTA: Este es un ejemplo educativo sin medidas de seguridad.
